## Project Details

This Teltonika EYE Sensors integration for Home Assistant was developed independently by Vignan Tej from Tez Solutions to provide seamless integration between Teltonika EYE sensors and Home Assistant.

## Acknowledgments

- **Home Assistant Community**: For the excellent platform and development guidelines
- **Bleak Library**: For providing robust Bluetooth LE support in Python
- **Teltonika**: For creating the EYE sensor hardware and protocol documentation
- **HACS Community**: For making custom integrations easily accessible

## Technical Implementation

The integration implements the complete Teltonika EYE sensor BLE protocol including:
- Temperature and humidity monitoring
- Movement detection and counting
- Magnetic field detection
- Battery monitoring and alerts
- Device orientation tracking
- Signal strength monitoring
